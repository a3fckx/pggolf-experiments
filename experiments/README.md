# Parameter Golf Experiments

This folder is the code mirror for experiment harnesses and runner scripts.

The vault remains the home for results and coordination at `a3fckx/experiments/`. This repo is where we should make code changes going forward.

## Start Here

- Start with `train_gpt.py` for the base harness.
- Use the phase folders for concrete experiment snapshots.
- Use the shell scripts at this level for launch, sync, and monitoring helpers.
- Use the public docs in `../docs/` for the shareable summary layer.

## What Lives Here

- code snapshots
- runner scripts
- experiment configs
- launch manifests

## What Does Not Live Here

- logs
- checkpoints
- plots
- result summaries
- coordination notes
